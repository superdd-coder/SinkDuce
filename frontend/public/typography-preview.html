Dark mode

How to review
  1. Section 01 — each token card shows size + family + sample text. Click into any token name to see real usage in components below.
  2. Section 02 — vertical rhythm: check whether the size steps between tiers feel right (not too tight, not too sparse).
  3. Section 03 — real components (sheet, dialog, chat input, sidebar, toast). If something feels off, that's the token to tune.
  4. Section 04 — verify dark mode contrast and family legibility (Source Serif 4 italic reads slightly differently on dark — flag if so).

To change a token, edit the value in the :root block at the top of this file. The page re-renders via native CSS — no reload needed.
